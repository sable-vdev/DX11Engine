# DX11Engine

DirectX 11 Render Engine created in C++ using imgui, stb, assimp.

Lots of implementations are made with the base of rastertek's online tutorial on directx11(https://rastertek.com/tutdx11win10.html)
Future goal: making a game engine based on this render engine

Current Features:
1. render queue submitting
2. Objectloading
3. Camera movement
4. texture loading, shaders
5. basic lightning per phong

Planned Features:
1. pbr implementation
2. better overlay with imgui
3. multithread support
